So what do media queries do?

Media queries allow users to change how a website styles objects depending on certain characteristics. One of the most common uses of media queries is for changing the layout of a website depending on what device you are viewing it from. It can also help increase accessibility by hiding parts that wouldn't be useful for those using screen readers or other accessibility tools.

When should I use media queries?

Media queries should be used whenever accessibility and consistency is desired (which should always be the case). It should be used when you want your content to have a consistent look and feel.

How should I use media queries?

Using media queries requires understanding the content you're presenting. Media queries should not conform to the latest hardware due to the constant changes. Media queries should be used as a tool to present your information in the best way possible on whatever medium it is on.

Some of the best situation for media queries include the following:

  • Style your content for mobile screens.
  • Style your content for printers. This is a great time to use them to hide banners, ads, and other unnecessary information.
  • When parts of your content would make your site harder to use for those using accessibility tools.

In closing, media queries should be used to make your content look good, be accessible, and look professional.
