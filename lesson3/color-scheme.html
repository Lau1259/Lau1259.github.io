Color Choices

The primary color for the web page will be a dark blue. This blue will be used in the navigation as text or on hover. The background of the header will be white. In contrast the footer will be dark blue and use white text. Link in this section should be green, orange or yellow; however, only one color should be used for all link in the footer. As for the main body, the background will be white with black text. The colors will be used to guide the users attention using color coded cards for specific content such as dark blue for local activities, orange for heavy traffic, green for outdoor activities, and yellow for mild delays.
